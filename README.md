# FazendoTibia para amigos

Baixe o jogo pela pagina **Releases** deste repositorio.

## Como jogar

1. Baixe `FazendoTibia-Para-Amigos.zip`.
2. Extraia o arquivo.
3. Abra `CRIAR PERSONAGEM.bat` para criar sua conta e personagem.
4. Abra `JOGAR.bat` e entre com o e-mail e a senha cadastrados.

O mapa revelado e instalado automaticamente pelo botao `JOGAR.bat`.

O servidor precisa estar online para o cadastro e o jogo funcionarem.
